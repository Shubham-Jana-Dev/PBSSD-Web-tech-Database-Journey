2. Data Manipulation Language (DML)
=> Deals with data present in the table.
=> Not auto-committed.

DML Commands:
1. INSERT
2. UPDATE
3. DELETE
4. SELECT

Syntax: INSERT INTO table_name (column1, column2, column3, .....)VALUES
(value1, value2, value3, ....);

Syntax: Update table_name SET column1 = value1, column2 = vlaue2,... WHERE condition;
